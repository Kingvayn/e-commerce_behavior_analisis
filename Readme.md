# Анализ поведения пользователей в e-commerce (Machine Learning)

## 📌 Описание проекта

Проект посвящён анализу поведения пользователей в e-commerce и попытке предсказания факта покупки с использованием методов машинного обучения.

Основная сложность датасета — наличие вложенных структур: история действий пользователей представлена в виде массивов словарей.

---

## 📊 Данные

Датасет содержит информацию о действиях пользователей, включая:

* просмотры товаров
* покупки
* вложенные структуры событий (массивы словарей)

---

## ⚙️ Обработка данных

Основной акцент в проекте сделан на предобработке данных:

* Парсинг вложенных JSON-подобных структур
* Преобразование данных в табличный формат
* Очистка и нормализация данных
* Feature engineering (создание признаков на основе истории действий)

---

## 📈 Исследовательский анализ (EDA)

* Анализ паттернов поведения пользователей
* Выявление доминирующих закономерностей в покупках

---

## 🤖 Модель

Была обучена модель машинного обучения (Logistic Regression / Random Forest) для предсказания факта покупки.

Этапы:

* Разделение на обучающую и тестовую выборки
* Обучение модели с использованием scikit-learn
* Оценка качества модели

---

## ⚠️ Результаты и ограничения

В ходе работы было выявлено, что модель демонстрирует признаки **переобучения** и плохо обобщает данные.

Основные причины:

* небольшой объём датасета
* наличие простого доминирующего паттерна в данных

Модель в значительной степени воспроизводит этот паттерн и не показывает существенного улучшения по сравнению с базовыми подходами.

---

## 💡 Выводы

Основная ценность проекта:

* работа со сложными структурированными данными

Проект демонстрирует важность качества и объёма данных при решении задач машинного обучения.
